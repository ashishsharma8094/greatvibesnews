This is a technical news website made using reactjs as a part of assessment .
To open webpage :
1. Open terminal in hackernews folder.
2. Run npm i.
3. Run npm start.
   And the website will start on the mentioned domain (http://localhost:3000).
   This app make use of hackernews api (https://hn.algolia.com/api) and gives us latest news in technical world.
